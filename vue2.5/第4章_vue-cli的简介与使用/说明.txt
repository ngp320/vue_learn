## 1 安装

###安装vue之前需要安装vs2019(可通过360软件管家)配置好C++桌面环境(或者安装后 工具-->工具.. 再配置C++桌面环境)
###全局安装vue-cli
npm install --global vue-cli
###创建一个基于webpack 模板的新项目
vue init webpack todolist
###安装依赖，走你
cd todolist
npm run dev
